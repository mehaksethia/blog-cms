# 🚀 Blog CMS Dashboard

A modern full-stack **Blog Content Management System (CMS)** where users can create, manage, and delete their own posts through a clean and interactive dashboard.

---

## ✨ Features

- 🔐 User Authentication (Signup & Login)
- 🧑‍💻 Personal Dashboard
- 📝 Create Blog Posts
- 🗑 Delete Posts
- 🎨 Modern UI with animations
- ⚡ Fast & responsive design

---

## 🛠 Tech Stack

**Frontend**
- HTML
- CSS (Custom UI + Animations)
- JavaScript

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB

---

## 🚀 How to Run Locally

```bash
git clone https://github.com/mehaksethia/blog-cms.git
cd blog-cms
npm install
node server.js
